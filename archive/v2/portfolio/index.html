DCPU-16 Toolkit

The DCPU-16 Toolkit consists of an assembler, disassembler, and an emulator.

What is a DCPU-16?

The DCPU-16 is a fictional 16 bit processor created for the game 0x10c. The DCPU-16 has 16 bit words, 0x10000 words of RAM, 8 registers, and runs at 100kHz. The full spec is available here. There are several hardware devices designed for the DCPU-16, including a monitor, keyboard, and clock.

Emulator

The emulator is a Windows console application written in C. The emulator fully implements the DCPU-16 1.7 spec along with the LEM1802 monitor, generic keyboard, and generic clock. With the emulator you can configure what hardware is attached the DCPU, load a binary to execute, and start and stop emulation. Each monitor connected to the DCPU is displayed in a seperate window.

Assembler and Disassembler

The assembler takes a DCPU-16 assembly source file as input and outputs a binary file that is executable by the emulator. The assembler recognizes all 36 opcodes of the 1.7 spec as well as comments, labels, and the DAT directive.

The disassembler goes the other way, taking a binary file as input and outputing an assembly source file.
